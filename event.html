What are Event Capsules?

About Event Capsules

★Event Capsules
These are available only during special campaigns!
Event Tickets
Using one Event/Collaboration Ticket will draw one Event Capsule.
You may receive tickets as stage rewards, Rare Capsule prizes, or as Login Bonuses.

*Each event will provide different ways of getting Event Tickets.

★Get limited-edition Units and Items!
Draw at the Event Capsule machine to get your hands on Units and Items appearing only during that specific event!
Gather many Event Tickets, and make sure to collect all the special Units and Items only available during that Event!

*Each event campaign will provide a different set of special Units and Items.
**Please note that the Event Capsule screen will not display once the event period has passed, and Event Tickets will not be able to be used. Event Tickets may not be refunded or exchanged for Normal/Rare tickets.

*If you are unable to draw a Capsule, please move to an area with strong network signal and try again.
